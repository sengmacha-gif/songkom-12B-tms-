ភាគ I: គណនាលីមីត និងការវិភាគចលនា

ដោះស្រាយទម្រង់មិនកំណត់ 0/0 និង អនន្ត/អនន្ត

លំហាត់ A
$$A = \lim_{x \to -\frac{\pi}{2}} \frac{3 + 3\sin x}{\cos^2 x}$$

របៀបបំប្លែងពីជួរទី១ ទៅជួរទី២៖ ទាញលេខ 3 ជាកត្តារួមនៅភាគយក បាន $3(1+\sin x)$។ ចំណែកភាគបែងប្រើរូបមន្តមាស $\cos^2 x = 1 - \sin^2 x = (1-\sin x)(1+\sin x)$ ដើម្បីសម្រួលតួបញ្ហាចោល។

លំហាត់ B
$$B = \lim_{x \to 4} \frac{\sqrt{x} - 2}{x - 4}$$

របៀបបំប្លែងពីជួរទី១ ទៅជួរទី២៖ បំបែកភាគបែង $x - 4$ ទៅជាផលគុណកត្តាដោយប្រើរាងតួឯកលក្ខណៈ $a^2 - b^2 = (\sqrt{x})^2 - 2^2 = (\sqrt{x}-2)(\sqrt{x}+2)$ រួចសម្រួលវាចោល។

លំហាត់ C
$$C = \lim_{x \to +\infty} \frac{3 - x}{2x - 1}$$

របៀបបំប្លែងពីជួរទី១ ទៅជួរទី២៖ ចាប់ដឺក្រេធំជាងគេ $x$ ជាកត្តារួមទាំងលើទាំងក្រោម រួចសម្រួល $x$ ចោល។ កាលណា $x \to \infty$ នោះតួ $\frac{1}{x} \to 0$។

🎛️ អូសដើម្បីមើលតម្លៃ $x$ ខិតជិត៖ -1.50

តម្លៃលទ្ធផលគណនាស្វ័យប្រវត្តតាមចលនាដៃ៖ 1.50

ភាគ II: គណនាអាំងតេក្រាលកំណត់ និងផ្ទៃក្រឡា

ស្វែងរកអនុគមន៍ដើម (ព្រីមីទីវ) និងការគណនាចន្លោះព្រំដែន $[a, b]$

លំហាត់ I
$$I = \int_{0}^{1} (x^2 - x + 2) dx = \left[ \frac{x^3}{3} - \frac{x^2}{2} + 2x \right]_{0}^{1} = \frac{11}{6}$$

បន្ទាប់ពីការបូកបង្រួមកត្តារួចរាល់ យើងទាញរកអនុគមន៍ដើមតាមរូបមន្តស្វ័យគុណ $\int x^n dx = \frac{x^{n+1}}{n+1}$ រួចជំនួសព្រំដែនលើដកព្រំដែនក្រោម។

លំហាត់ J
$$J = \int_{2}^{3} \left( \frac{1}{x-1} + \frac{1}{x^2} \right) dx = \left[ \ln|x-1| - \frac{1}{x} \right]_{2}^{3} = \ln 2 + \frac{1}{6}$$

អនុគមន៍ដើមនៃ $\frac{1}{x-1}$ គឺ $\ln|x-1|$ ហើយតួ $\frac{1}{x^2} = x^{-2}$ នាំឱ្យអនុគមន៍ដើមរបស់វាគឺ $\frac{x^{-1}}{-1} = -\frac{1}{x}$។

លំហាត់ K
$$K = \int_{0}^{\ln 2} (2e^{2x} + 3e^x) dx = \left[ e^{2x} + 3e^x \right]_{0}^{\ln 2} = 6$$

ដោយសារអនុគមន៍ដើមនៃ $e^{ax}$ គឺ $\frac{1}{a}e^{ax}$ ដូចនេះ $2e^{2x}$ ខ្លាយជា $2 \cdot \frac{1}{2}e^{2x} = e^{2x}$។ ពេលជំនួស $\ln 2$ ចូលយើងសម្រួលតាមលក្ខណៈអិចស្ប៉ូណង់ស្យែល។

ភាគ III: ការវិភាគប្រូបាប និងគំរូស្ថិតិ

ការទាញយកកន្សោមករណីអាច $C(n, r)$ និងករណីស្រប

📦 លក្ខខណ្ឌបម្រាប់ប្រធានលំហាត់៖

4 ក្បាល 🐟 ត្រីមានស្រកា
6 ក្បាល 🐡 ត្រីអត់ស្រកា
🎯 ចាប់ព្រមគ្នា 3 ក្បាល ⇒ ករណីអាច៖ $C(10,3) = 120$ របៀប

🟢 ព្រឹត្តិការណ៍ A (ត្រីស្រកាទាំង 3): រើសត្រីស្រកា 3 ពីក្នុងចំណោម 4 នាំឱ្យបាន $C(4,3) = 4$ របៀប។ ប្រូបាបគឺ $P(A) = \frac{4}{120} = \frac{1}{30} \approx 0.0333$

🔵 ព្រឹត្តិការណ៍ B (អត់ស្រកា 2 និងស្រកា 1): រើសត្រីបាន $C(6,2) \times C(4,1) = 15 \times 4 = 60$ របៀប។ ប្រូបាបគឺ $P(B) = \frac{60}{120} = \frac{1}{2} = 0.50$

ភាគ IV: សមីការនិងក្រាហ្វិកអេលីប 2D/3D

បំប្លែងទៅជាសមីការស្តង់ដារ រកកំពូល និងកំណុំ

🔄 ការបំប្លែងសមីការ៖

ពី $12y^2 = 300 - 75x^2$ បោះតួ $x$ មកឆ្វេង រួចចែកនឹង 300 ទទួលបានទម្រង់ស្តង់ដារ៖ $$\frac{x^2}{2^2} + \frac{y^2}{5^2} = 1$$

📐 ធាតុសំខាន់ៗ៖
  • ពាក់កណ្តាលអ័ក្សធំ៖ $a = 5$ (ឈរ)
  • ពាក់កណ្តាលអ័ក្សតូច៖ $b = 2$ (ដេក)
  • ចម្ងាយផ្ចិតទៅកំណុំ៖ $c = \sqrt{25-4} = \sqrt{21}$
📍 កូអរដោនេ៖
  • កំពូល៖ $V_1(0,5), V_2(0,-5)$
  • កំណុំ៖ $F_1(0,\sqrt{21}), F_2(0,-\sqrt{21})$

ភាគ V: សិក្សាអនុគមន៍សនិទាន និងអាស៊ីមតូត

ដែនកំណត់ អាស៊ីមតូតឈរ $x=1$ និងអាស៊ីមតូតទ្រេត $y=x+1$

🔍 ដែនកំណត់ និងការបំបែក៖

ដែនកំណត់៖ $D = \mathbb{R} \setminus \{1\}$ ព្រោះភាគបែង $1-x \neq 0$។

ការបំប្លែងរកអាស៊ីមតូតទ្រេត តាមរយៈការតម្រូវភាគបែងឡើងវិញ៖

x + 1 + 3/(1-x) = (4 - x²)/(1-x)

💡 គន្លឹះគូរក្រាហ្វិក៖

ខ្សែោងស្អិតជាប់ត្រង់ $x=1$ ត្រូវបានកាត់ផ្តាច់ចេញជាពីរចំណែក (Left & Right branches) ដោយសារតម្លៃលីមីតខិតជិតត្រង់ $1^-$ ស្មើ $+\infty$ ហើយត្រង់ $1^+$ ស្មើ $-\infty$។ បន្ទាត់ពណ៌ប្រផេះដាច់ៗគឺជាខ្សែអាស៊ីមតូតទ្រេតដែលក្រាហ្វិកដេញតាមមិនចេះទើស។
